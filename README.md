# PS Catering & Events Management Website

This is a modern event management website created for **PS Catering & Events Management, Madurai**.

## Features

- Responsive design
- Wedding & event service showcase
- Catering menu section
- Event gallery
- Client testimonials
- Contact and WhatsApp booking

## Built With

- React
- Vite
- Tailwind CSS
- Framer Motion

## Sections

- Hero
- About
- Awards
- Services
- Catering Menu
- Gallery
- Testimonials
- Contact
